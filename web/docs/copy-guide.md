# Soak 文案指南

中文移动端 PWA(控糖 / 抗炎 / 中医语言)。所有 UI 文案统一遵守以下规则。

## 1. 音调:口语温和,不端着

✅ "今天还没打卡"
❌ "您今日尚未完成打卡"(机器翻译腔)

✅ "再坚持解锁完整版"
❌ "持续 21 天即可解锁完整版本"(说明书腔)

✅ "可能引起反应,注意吃多"
❌ "本食材可能引发不良反应,请注意摄入量"(医嘱腔)

**禁用词**:您、敬请、谨慎、烦请、请注意。
**允许词**:你、记得、试试、可以。

## 2. 量词表(按食物种类)

| 食物 | 量词 | 动词 | 例 |
|------|------|------|----|
| 茶饮(奶茶/咖啡/汽水) | 杯 / 罐 | 喝 | 少喝 1 杯奶茶 |
| 巧克力 / 糖块 | 块 | 吃 | 少吃 2 块巧克力 |
| 棒棒糖 | 根 | 吃 | 少吃 3 根棒棒糖 |
| 餐 | 餐 / 张(指照片) | 拍 | 拍 1 张餐照 / 完成 2 餐 |
| 水(喝水) | 杯 | 喝 | 喝了 5 杯 / 0 / 8 杯 |
| 糖(总量) | g | 摄入 | 累计少摄入 76 g 添加糖 |

## 3. 中英文混排

**字符之间留半角空格**(中英 / 中数 / 中符号边界):

✅ `30 秒测一下你当前的抗炎指数`
✅ `累计 0 / 21 天`
✅ `本月减糖 76 g`
✅ `2026 年 5 月`

❌ `30秒测一下` / `累计0/21天` / `本月减糖76g`

例外:中文标点(逗号/句号/引号)前后**不加**空格;数字单位(g/kg/% 等)与数字之间**也不加**(`100%` / `45g` 写法可读性更高,但本项目统一用 `45 g` 风格,见上)。

## 4. 标点

- **句号**:中文 `。`,不用英文 `.` 或省略。
- **逗号**:中文 `,`,不用 `,`。
- **冒号**:中文 `:`(全角),不用 `:`。
- **顿号**:用 `、` 列举(食材列表),不用 `,`。
- **引号**:中文 `「」` 或 `""`,不混用。引文献 / 来源用 `《》`(典籍)或 `[]`(论文)。
- **省略号**:`…`(单个 unicode),不用 `...`(3 个英文点)或 `。。。`。
- **数字 / 字母列表分隔**:用 ` · `(中点)而不是 `,`,例:`燕麦 · 核桃 · 蓝莓`
- **范围**:`1 - 5` 用 `~`(`1~5`)或 `-`(`1-5`),保持每页一致即可。本项目优先 `-`。

## 5. 鼓励语 / 评价语

按今日 tier 分档,1 句一段,**不超过 18 字**:

| Tier | 调性 | 例 |
|------|------|----|
| perfect (完美一天) | 庆祝 | 今天稳得很 ✨ |
| great (美好一天) | 鼓励 | 节奏不错,继续保持 |
| nice (奈斯一天) | 温和 | 今天稍稍踩了点,明天再调 |
| none (未达成) | 中性 | 今天还没开始,先拍一餐试试看 ✨ |

火分 4 档(平 / 微火 / 中火 / 大火)对应等级标签:

| 火分 | 标签 | 评价语 |
|------|------|--------|
| 平 | 轻盈 ★★★★★ | 整体很清气,继续保持 |
| 微火 | 微暖 ★★★★ | 略温,多喝水多休息 |
| 中火 | 留心 ★★★ | 有点上火苗头,清淡几餐 |
| 大火 | 注意 ★★ | 火气重,饮食先收一收 |

## 6. 错误 / 异常文案

不甩技术细节给用户,但要给出**下一步行动**:

✅ "网络不通,稍等再试"
❌ "Failed to fetch: ECONNREFUSED"

✅ "看不太清,要不要补一张?"
❌ "识别置信度低于阈值"

✅ "服务忙,请稍后再试"
❌ "503 Service Unavailable"

## 7. 量化数据呈现

- 优先**整数**;有小数时保留 1 位(`减糖 35.5 g`)
- 大数字用千分位(暂未启用,统一不加千分位也可)
- 百分比用整数(`60%` 不写 `60.0%`)
- 时间:`5 月 7 日 星期四` / `2026.05`(月份切换器极简)/ `30 秒` / `1.5 秒`(技术阈值才用秒)

## 8. CTA 按钮文案

主按钮一律用**短动作 + 期待结果**(动词开头,不超过 8 字):

✅ "拍 / 选这一餐照片"
✅ "现在就拍第一张 →"
✅ "30 秒测一下"
❌ "提交" / "确认" / "OK"(过于干瘪)

次按钮可用**无方向性短语**:

✅ "稍后再拍"
✅ "再拍一张"
✅ "回主页"
